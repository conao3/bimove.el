Binary search-based cursor movement for Emacs.

[[./img/bimove.gif]]

* Overview

bimove.el provides an efficient way to navigate within the visible buffer using a binary search approach. Instead of scrolling or jumping line by line, you can quickly narrow down to any visible line by repeatedly halving the search range.

Inspired by [[https://github.com/kamecha/bimove][kamecha/bimove]].

* Installation

** Using package-vc (Emacs 29+)

#+begin_src emacs-lisp
(package-vc-install "https://github.com/conao3/bimove.el")
#+end_src

** Using leaf.el

#+begin_src emacs-lisp
(leaf bimove
  :vc (:url "https://github.com/conao3/bimove.el")
  :ensure t)
#+end_src

* Usage

Activate bimove-mode to start binary search navigation:

#+begin_src emacs-lisp
M-x bimove-mode
#+end_src

Or bind it to a key for quick access:

#+begin_src emacs-lisp
(global-set-key (kbd "C-c b") #'bimove-mode)
#+end_src

** Navigation Keys

Once active, the visible window is divided into three colored regions:

| Key       | Action                                    |
|-----------+-------------------------------------------|
| ~p~       | Move to the high range (upper half)       |
| ~n~       | Move to the low range (lower half)        |
| ~q~ / ~RET~ | Exit and stay at the current position  |

** Visual Indicators

- *High range* (cyan) - upper half of the search area
- *Mid line* (yellow) - current cursor position
- *Low range* (red) - lower half of the search area

* Customization

Adjust the highlight colors to match your theme:

#+begin_src emacs-lisp
(set-face-attribute 'bimove-high-face nil :background "#BADFDB" :extend t)
(set-face-attribute 'bimove-mid-face nil :background "#FCF9EA" :extend t)
(set-face-attribute 'bimove-low-face nil :background "#FFA4A4" :extend t)
#+end_src

Or use the built-in customization interface:

#+begin_src
M-x customize-group RET bimove RET
#+end_src
